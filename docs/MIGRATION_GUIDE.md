# 迁移指南

本文档列出从较旧版本迁移到 v0.2.8 的重要注意点。

1. `Detector.detect` 签名
   - 新增可选参数 `categories`（list）。
   - 推荐在配置或调用时显式传入 `categories` 以便框架层面过滤结果。

2. 示例与文档位置
   - `examples/` 已按功能重构，常用示例位于 `examples/` 根目录。

3. 测试与可选依赖
   - 若你的 CI 在测试时出现导入错误，请采用延迟导入可选后端（例如 ultralytics/torch），或在测试环境安装相应可选依赖。

4. 向后兼容
   - 所有原有 API 保持兼容，`categories` 为可选参数，不会改变默认行为。

如果遇到具体问题，请附带最小重现示例提交 Issue。
